Web dizajn

Web dizajn je proces planiranja, kreiranja i uređivanja web stranica. Uključuje raspored elemenata, izbor boja, tipografiju i prilagođavanje različitim uređajima. Dobro dizajnirana stranica nije samo lijepa, već i funkcionalna i laka za korišćenje.

1. Primjer neuredjene liste:

Web dizajn uključuje:

  • Planiranje
  • Kreiranje
  • Uredjivanje

2. Primjer uredjene liste:

Proces kreiranja web stranice:

  1. Napravi HTML fajl
  2. Povezi CSS (Cascade style sheet)
  3. Dodaj sadrzaj sajta
  4. Sredi dizajn elemenata

3. Primjer ugnijezdene liste:

Da bi se bavio front-end razvijanjem, moras poznavati sledece jezike:

  • HTML
    • Bootstrap
    • Tailwind
  • CSS
    • Boje
    • Fontovi
    • Pozadina
  • JavaScript
    • Vue.js
    • React.js
    • AngularJS
    • TypeScript (tipiziran JS)
